# Plan

This document tracks what the repo currently does and what could be added or modified. Update this when capabilities or goals change.

## What this repo does
- Provides configuration for shells, editors, terminal apps, and related tools.
- Supports setup on macOS and Linux.
- Includes scripts to bootstrap or apply configuration.

## Areas covered
- Shell configuration (bash and related).
- Terminal emulators (Alacritty, iTerm).
- Terminal multiplexer (tmux).
- Editor configuration (Neovim).
- OS-specific setup scripts (macOS, Linux).

## What could be added
- Zsh configuration (if desired).
- Additional editor configs (e.g., VS Code, Emacs).
- More OS-specific automation (Linux distributions, WSL).
- Modularization of shared config across OSes.
- Automated verification or linting for config files.

## What could be modified
- Simplify or consolidate overlapping config.
- Improve documentation for setup and troubleshooting.
- Add version checks or guards in setup scripts.
- Better separation between personal and shared settings.

## Next review
- Update after significant config changes or new tooling.
